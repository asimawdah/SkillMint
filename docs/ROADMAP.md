# Roadmap

## v0.1

- One-command interactive flow.
- Detect Flutter, React, Laravel, Python, FastAPI, Django, Go, Docker, and GitHub Actions.
- Generate AGENTS.md, CLAUDE.md, Cursor rules, and Copilot instructions.
- Install Flutter official agent skills from GitHub with confirmation.
- Generate local skills for supported stacks.

## v0.2

- Better package manager detection for npm, pnpm, yarn, uv, poetry, and pipenv.
- More framework-specific rules for Next.js, Vue, Nuxt, Svelte, NestJS, Express, Rails, Rust, and Kubernetes.
- Safer overwrite strategy with diffs/backups.
- Optional machine-readable output for agent integrations.

## Later

- Public skill registry.
- Signed/trusted skill metadata.
- Plugin system for custom team skills.
- Project-specific convention learning from existing files.

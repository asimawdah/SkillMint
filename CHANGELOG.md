# Changelog

## 0.1.0

- Initial private MVP scaffold.
- One-command interactive flow through `skillmint`.
- Stack detection for Flutter, React, Laravel, Python, FastAPI, Django, Go, Docker, and GitHub Actions.
- Generates AGENTS.md, CLAUDE.md, Cursor rules, and GitHub Copilot instructions.
- Installs external Flutter official skills with user confirmation.
- Generates local fallback skills for other stacks.
